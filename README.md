# BU_Check
2018년도 부천대학교 컴퓨터소프트웨어학과 C#프로그래밍 프로젝트   
BU_Check - 출석체크 프로그램   
제작 기간 : 2018/09 ~ 2018/12    

## Team Members
* [ymiru0324(김용민)](https://github.com/ymiru0324)   
* 김현우   
* 김동현   

## 프로젝트 개요
* 자동 출석체크 프로그램으로 편의성 향상   
* 출석 내용에 대한 간단한 정보 유지 · 관리 · 열람   
* 하나의 프로그램에서 학생의 출석 현황과 성적을 동시에 관리 및 확인
